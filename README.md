# ♟ Chess vs AI (WebSocket Engine)

A simple browser-based chess game where you can play against an AI engine using WebSockets.

## 🚀 Features

- Play as White or Black
- Real-time moves using WebSocket engine
- Draw and Resign options
- Automatic game reset
- Clean and responsive UI

## 🧠 Tech Stack

- HTML, CSS, JavaScript
- chess.js (game logic)
- chessboard.js (UI board)
- WebSocket (AI engine)
